Cliff Diving Game

Justin Brunings - CSC 474 - Spring 2023 - Christian Eckhardt

Project Description

The goal of the project was to recreate a famouse 2d cliff diving game

I implemented skinning animations and skinning blending for the diver, as well as particles when entering the water. The particles have different effects depending on how the diver enters the water.
